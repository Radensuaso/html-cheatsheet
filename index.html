Basic HTML

Basic tags

Tag Name Description
<!DOCTYPE html> Document Type Declaration Defines the document type
<html> HTML Element Defines an HTML document
<head> Head Element Contains metadata/information for the document
<title> Title Element Defines a title for the document
<body> Body Element Defines the document's body
<h1> to <h6> Heading Elements Defines HTML headings
<p> Paragraph Element Defines a paragraph
<br> Line Break Element Inserts a single line break
<!--...--> Comments Defines a comment

Basic Attributes

Attribute Name Description
class="" Class Attribute One class can be assigned to multiple elements
id="" ID Attribute One ID can be assigned to only one element
